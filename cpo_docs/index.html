Authoring Workflow

  1. Draft release text.
  2. Choose subtype template; add JSON-LD.
  3. Register claims; attach evidence.
  4. Run linter locally; fix issues.
  5. Commit & open PR; CI validates.
  6. Publish to site; update registry.
